# Polito-Indoor-Navigation-Challenge2025

**InMaps** is an indoor–outdoor integrated navigation system designed to improve campus navigation efficiency at Politecnico di Torino.  
The project was developed as part of **Challenge@PoliTO 2025** by Team 2 (InMaps).  
It addresses major issues of the existing PoliTO Map and provides a smart routing experience with cross-floor, multi-building mobility support.

##  Project Goals

- Help students, faculty and visitors find destinations quickly and efficiently
- Enable **indoor–outdoor seamless navigation**
- Support **cross-floor routing**, intelligent map switching, and accessibility
- Provide **personalized navigation** using identity-based schedule integration
- Enhance campus experience through improved UI/UX
